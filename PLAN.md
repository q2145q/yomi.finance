# План разработки yomi.finance

Обновлено: 2026-02-25

## Этап 1 — Ядро (текущий)

- [ ] Авторизация и ролевая модель (JWT, 5 ролей)
- [ ] Управление проектами (CRUD, статусы)
- [ ] Управление контрагентами (карточка, типы ФЛ/СЗ/ИП/ООО)
- [ ] Налоговые схемы (СЗ, ИП, НДС, ФЛ, ИП+НДС)
- [ ] Модуль Бюджет — структура и вложенность статей
- [ ] Модуль Бюджет — расчёт налогов
- [ ] Модуль Бюджет — экселевидный интерфейс (Handsontable)
- [ ] Шаблон бюджета (16 категорий, 47 подкатегорий, 182 строки)
- [ ] Docker Compose (db + backend + frontend/nginx)
- [ ] Деплой на finance.yomimovie.art

## Этап 2 — Производство

- [ ] Импорт КПП (xlsx)
- [ ] Договоры (привязка к статьям бюджета)
- [ ] Производственный отчёт (ручной ввод)
- [ ] Табели (представление ReportEntry по договору)

## Этап 3 — Финансы

- [ ] Заявки на оплату (оклад / посменные / счета)
- [ ] Загрузка платёжек и закрытие платежей
- [ ] Кэшфлоу (плановый + фактический)

## Этап 4 — Автоматизация

- [ ] Telegram-бот (регистрация исполнителей, отчёты смен, загрузка чеков)
- [ ] AI-парсинг (Claude API, confidence threshold 0.7)
- [ ] Уведомления и автотриггеры
